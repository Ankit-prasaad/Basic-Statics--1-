# Basic-Statics-1

The descriptive analysis shows that the dataset has high variation in sales-related features such as Average Price, Total Sales Value, Discount Amount, and Net Sales Value. Some variables contain outliers, which represent unusually high-value transactions. Most numerical features are right-skewed, where lower values occur more frequently than higher values. The Discount Rate (%) remains relatively stable compared to other variables. The categorical analysis also shows that some categories appear more frequently than others. Overall, the analysis helps in understanding the data pattern and preparing the dataset for further preprocessing and model building.

EDA is also done inside the data(sales_data_with_discounts) for the better insights.
